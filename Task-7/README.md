# TaskFlow - DOM Mastery Edition

A fully interactive Task Manager built with **pure HTML, CSS, and Vanilla JavaScript** to demonstrate core browser concepts.

## Features

- Add, Edit, Delete, and Complete tasks
- Event Delegation on task container
- Event Bubbling & Capturing demonstration
- Dark/Light Mode toggle using `data-theme`
- Search and Filter functionality
- Local Storage persistence
- Visual Browser Rendering Pipeline explanation
- Attributes vs Properties demonstration (check console)

## Learning Objectives Covered

- **DOM Manipulation**: `createElement`, `appendChild`, `remove`, `dataset`, etc.
- **Event Handling & Delegation**
- **Event Propagation** (Bubbling vs Capturing)
- **Attributes vs Properties**
- **Browser Rendering Pipeline** (HTML → DOM Tree, CSS → CSSOM, Render Tree)

## How to Run

1. Download all files into one folder:
   - `index.html`
   - `style.css`
   - `script.js`
2. Open `index.html` in your browser.

## Deployment

Deploy easily on **Netlify**, **Vercel**, or **GitHub Pages**.
